Ниже — максимально подробный разбор дизайна приглашения по изображению. Часть параметров (точные шрифты, размеры и интервалы) невозможно определить на 100% без исходного макета, поэтому я укажу наиболее вероятные аналоги и приблизительные значения.

# Общая концепция

**Стиль:** Modern Editorial Wedding / Minimal Luxury / Vogue-inspired

Основная идея построена на сочетании:

* высокой типографики;
* большого количества воздуха;
* монохромных фотографий;
* контраста между строгими засечками и рукописным каллиграфическим шрифтом;
* нейтральной премиальной палитры.

Ассоциации:

* свадебные журналы Vogue Bride;
* editorial fashion;
* luxury minimalism;
* европейская дворцовая свадьба.

---

# Формат композиции

Макет разделен на **2 вертикальные колонки**.

Пропорции примерно:

* левая колонка: 55–58%
* правая колонка: 42–45%

Между колонками:

* вертикальная бежевая полоса
* ширина около 4–5% макета

---

# Цветовая палитра

Основные цвета:

### Фон

Светло-молочный

Примерно:

```css
#F7F7F5
```

или

```css
#F4F3F0
```

---

### Вертикальная разделительная полоса

Теплый серо-бежевый

Примерно:

```css
#D4CEC6
```

или

```css
#CEC7BF
```

---

### Текст

Темно-серый

Не чистый черный.

Примерно:

```css
#2E2E2E
```

---

### Акцентный цвет

Используется только в палитре дресс-кода.

Палитра:

1. Молочный

```css
#E8DFD2
```

2. Бежевый

```css
#CFC2B0
```

3. Серо-коричневый

```css
#6B6968
```

4. Черный

```css
#000000
```

---

# Типографическая система

Используется минимум 4 разных семейства шрифтов.

---

# Шрифт №1 (главный декоративный LOVE)

Очень тонкий Didone Serif.

Похожие:

* Didot
* Bodoni Moda
* Cormorant Garamond Light
* Bodoni 72
* Glamour Absolute

Наиболее вероятно:

**Didot**

Особенности:

* высокий контраст штрихов;
* тончайшие волосные линии;
* длинные вертикали.

---

# Шрифт №2 (имена, дата, таймер)

Тонкий fashion serif.

Похожие:

* Cormorant Garamond
* Cormorant Infant
* Baskerville Display
* Bodoni Moda

Для имён:

```
MARTIN & ALENA
```

очень похоже на:

**Cormorant Garamond Light**

---

# Шрифт №3 (каллиграфия)

Используется для:

* Love governs the world
* Приглашаем
* Место
* Дресс-код

Вероятные аналоги:

* Bickham Script
* Edwardian Script
* Burgues Script
* Citadel Script
* Great Vibes (более современный аналог)

Наиболее близко:

**Bickham Script Pro**

Особенности:

* длинные росчерки;
* высокая контрастность;
* свадебный характер.

---

# Шрифт №4 (основной текст)

Тонкий современный гротеск.

Похожие:

* Montserrat Light
* Raleway Light
* Gotham Light
* Futura PT Book

Вероятнее:

**Montserrat Light**

Используется для:

* описания приглашения;
* адреса;
* подписей таймера.

---

# Левая колонка

## Верхняя карточка

Белый прямоугольник со скруглением.

Примерно:

```css
border-radius: 12px
```

Отступ сверху:

≈ 40 px

Отступ слева:

≈ 30 px

---

## Имена

Расположение:

по центру.

Размер:

примерно:

```css
20–24 px
```

Межбуквенный интервал:

```css
0.08em
```

---

## LOVE

Главный фокус макета.

Особенности:

* вертикальная композиция;
* буквы наслаиваются друг на друга;
* занимает около 40% высоты карточки.

Размер:

примерно:

```css
140–180 px
```

---

## Вертикальная надпись

Слева:

```
Love governs the world
```

Разворот на 90°.

Шрифт:

каллиграфия.

Размер:

≈ 30–36 px

---

## Дата

```
26.10.2025
```

Выравнивание:

по левому краю.

Размер:

≈ 24–28 px

---

## Заголовок

```
Приглашаем
```

Каллиграфический шрифт.

Размер:

≈ 40 px

---

## Основной текст

Центрирован.

Размер:

≈ 12–14 px

Межстрочный интервал:

150%

---

## Большая дата внизу

```
26
10
25
```

Сложенная композиция.

Шрифт:

тот же Didot.

Размер:

≈ 90–120 px

# Что делает дизайн дорогим визуально

1. Очень тонкие засечки (Didot-стиль).
2. Много пустого пространства.
3. Черно-белая фотография.
4. Ограниченная палитра.
5. Минимум декоративных элементов.
6. Типографика как главный графический элемент.
7. Каллиграфия используется только как акцент.
8. Асимметричная журнальная верстка.
